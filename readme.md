Transporte Fundemap
Mapa interactivo de las paradas de los micros hacia la Planta Fundemap.

Funcionalidades
67 paradas de los micros 1 y 2.
Filtros independientes por recorrido.
Buscador por calle o parada.
Autocompletado de direcciones de Rosario desde cuatro caracteres.
Selección de una dirección con las diez paradas más cercanas.
Ubicación actual mediante GPS, con permiso del usuario.
Vistas separadas de paradas y mapa en celulares para aprovechar toda la pantalla.
Marcadores numerados y líneas de recorrido.
Diseño adaptable a computadoras y celulares.
Uso local
Abrir index.html en un navegador con conexión a internet. Leaflet y las capas de OpenStreetMap se cargan desde servicios públicos externos.

Publicación
El sitio es estático y puede publicarse directamente mediante GitHub Pages desde la rama main.

Fuentes
Las paradas provienen del cronograma de transporte provisto por Fundemap. Las coordenadas se calcularon a partir de OpenStreetMap; algunas ubicaciones pueden ser aproximadas.
